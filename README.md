# 🍫 ChocoCrunch Nutrition & Sales Analytics

## 🔎 Project Summary

Built an end-to-end data analytics project analyzing chocolate product nutrition and market patterns.
The project demonstrates skills in data cleaning, exploratory analysis, and dashboard development to generate business-ready insights for product positioning and decision-making.

---

## 🎯 Objectives

* Analyze calorie and sugar distribution across chocolate products
* Compare brand-level nutrition trends
* Identify patterns useful for product strategy and marketing decisions
* Build an interactive dashboard for quick business insights

---

## 🧠 Key Skills Demonstrated

* Data Cleaning & Feature Engineering
* Exploratory Data Analysis (EDA)
* Business Insight Generation
* Dashboard Development
* Data Storytelling

---

## 🛠 Tech Stack

* **Python:** Pandas, NumPy, Matplotlib
* **Visualization:** Power BI
* **Environment:** Jupyter Notebook
* **Dataset:** CSV structured product data

---

## 📊 Business Impact

* Highlights nutrition patterns affecting consumer perception
* Helps brands identify healthier product positioning opportunities
* Enables faster data-driven decisions using an interactive dashboard

---

## 📂 Repository Contents

* `choco_final.ipynb` → Data analysis workflow
* `choco_final.csv` → Processed dataset
* `Choco Power BI analysis.pbix` → Interactive dashboard
